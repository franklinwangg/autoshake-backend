Backend Software Engineer – New Grad (2025/2026)
Stripe

About the Role

We are looking for a new grad software engineer to join Stripe's backend infrastructure team. You will work on the systems that process hundreds of billions of dollars in payments annually, building the reliable, scalable services that millions of businesses depend on.

Responsibilities

- Design and build backend services and APIs that are reliable, scalable, and maintainable.
- Own features end-to-end from technical design through production deployment and monitoring.
- Write high-quality, well-tested code and participate in code review.
- Debug and resolve production issues with a focus on root cause analysis.
- Work cross-functionally with product, data, and infrastructure teams.

Requirements

- Bachelor's degree in Computer Science or a related field, graduating in 2025 or 2026.
- Strong programming fundamentals in at least one backend language (Ruby, Go, Java, Python, or Scala).
- Solid understanding of data structures, algorithms, and system design.
- Experience with relational databases (PostgreSQL, MySQL) and writing efficient SQL.
- Ability to work in a fast-paced, collaborative environment.

Nice to Have

- Experience with distributed systems concepts (consensus, replication, partitioning).
- Familiarity with cloud infrastructure (AWS, GCP).
- Experience with containerization and CI/CD pipelines (Docker, GitHub Actions).
- Prior internship experience at a technology company.
- Experience working with large-scale data pipelines or ETL systems.

Why Stripe

Stripe is a global technology company that builds economic infrastructure for the internet. We process payments for companies ranging from early-stage startups to public companies like Amazon, Google, and Salesforce. Our backend engineers work on some of the hardest distributed systems problems in production.
